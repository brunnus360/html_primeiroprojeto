O que são Client

Voltar

Client ou Cliente são os computadores que fornecem a interface do usuário e de onde os usuários podem solicitar serviços ao servidor.

Essas solicitaçoes de serviço podem ser por exemplo acesso a dados, carregamento de páginas web, acesso a aplicações ou receber emails.

Por exemplo um site é hospedado em um servidor que será acessado por um cliente por meio de um navegador.

Esse acesso é por meio de uma solicitação ao servidor que envia de volta para o navegador os arquivos do site que são HTML,CSS, JavaScript e Midias que serão executados por ele.

Tendo como resultado exibido na tela o site a ser navegado pelo usuário.

O que são Server

Voltar

Server ou Servidor são os computadores que fornecem serviços aos Clients.

Esses serviços podem ser de naturezas distintas, como por exemplo, arquivos e emails.

São computadores robustos e caros pois necessitam ter muita capacidade de processamento para atender a vários clientes.

Possui um sistema operacional especifico para ele como por exemplo o Windows Server, Ubuntu, Debian e outros.

Alguns tipos de servidores são:

  • Proxy: possui regras para acesso a internet; funciona como um servidor de autenticação
  • Segurança: voltado para o acesso externo
  • Web: Gerencia as requisições HTTP
  • Email: Gerencia correio eletrônico

Falando Sobre Linguagem de Programação

Voltar

A Linguagem de Programação é um conjunto de instruções e regras para gerar programas.

Serve de meio de comunicação entre computadores e humanos por isso é escrita em linguagem escrita e formal.

Segundo o GitHub em seu relatório Octoverse 2022, as 5 linguagens mais usadas em 2022 foram:

  1. JavaScript
  2. Python
  3. Java
  4. TypeScript
  5. C#

As linguagens Server-Side são as linguagens executadas no servidor mais popularmente conhecidas como linguagens back-end.

Alguns exemplos são JavaScript, PHP, Ruby, Python, Java e outras.

As linguagens Client-Side são as linguagens que são executadas no lado cliente.

JavaScript é a mais usada.

O JavaScript é a linguagem Client-Side mais utilizada.
